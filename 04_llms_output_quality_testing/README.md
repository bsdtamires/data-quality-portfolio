# Project LLM Output Quality Testing - DeepEval + PromptFlow + Snowflake

🚧 Work in Progress 🚧

## Goal
Comprehensive quality testing framework for Large Language Models (LLMs) and generative AI applications. This project demonstrates best practices in evaluating LLM outputs, managing prompt engineering workflows, and storing evaluation results in Snowflake for analysis and continuous improvement.
