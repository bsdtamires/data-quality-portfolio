# Data Quality Portfolio

A comprehensive portfolio showcasing modern approaches to data quality testing, and validation across different tools, technologies, and use cases. These projects demonstrate best practices in data quality engineering/quality assurance and AI-assisted methodologies.

## 📚 Projects Overview

| Project | Description | Technologies | Status |
|---------|-------------|--------------|--------|
| [project01_SQL_Testing_AI_dq](./project01_SQL_Testing_AI_dq/README.md) | SQL+ AI-assisted test data & database-generated data quality framework | SQL, AI/LLM, Data Generation | 🚧 WIP |

| [project03_dagster_ai_quality_pipeline](./project03_dagster_ai_quality_pipeline/README.md) | Orchestrated DQ pipelines with AI-powered monitoring & intelligent alerting | Dagster, AI/ML, Orchestration | 🚧 WIP |
| [project04_LLM_output_quality_testing](./project04_LLM_output_quality_testing/README.md) | DeepEval + PromptFlow + Snowflake for LLM output quality evaluation | DeepEval, PromptFlow, Snowflake, LLM | 🚧 WIP |

## 🎯 Portfolio Mission

This portfolio demonstrates:

- **Data Quality Best Practices**: Comprehensive approaches to ensuring data reliability, completeness and accuracy
- **Modern Tech Stack**: Integration of contemporary tools (Dagster, dbt, Great Expectations, DeepEval, PromptFlow)
- **AI-Assisted Development**: Leveraging LLMs for intelligent test generation, monitoring, and quality improvement
- **Scalable Architectures**: Production-ready patterns for data quality at enterprise scale
- **Multiple Paradigms**: Various approaches to quality validation (SQL, dbt, Python, orchestration, LLM evaluation)
